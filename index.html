BEBASID

BEBASID dapat membantu membuka halaman website yang diblokir Internet Positif dengan memanfaatkan hosts.

Bebas Virus

Kalian bisa menginstallnya secara manual atau dengan cara menggunakan versi GUI yang tentu saja sangat aman dari virus.

Jangkauan Situs

BEBASID menjangkau cukup banyak situs, kalian bisa menambahkannya dengan cara membuka issue baru di GitHub maupun di Discord.

Dukungan Netflix

Jika ISP kamu mempunyai masalah dengan platform hiburan Netflix, kami bisa membantu menyelesaikan permasalahan yang kamu alami.

Prosedur & Cara Kerja

1

File hosts adalah file computer yang digunakan oleh sistem operasi untuk memetakan hostname ke alamat IP.

2

Tim BEBASID membuat peta alamat IP dan hostname secara rinci dan melalui proses percobaan.

3

Hasil file hosts yang sudah dikerjakan tim akan diunggah dan disediakan ke publik.

4

File hosts diunduh oleh pengguna dan dipasang pada sistem akan menggantikan hosts lama.

5

Sistem akan mendeteksi adanya hosts baru dan akan mulai memetakan halaman ke peta baru.

6

Halaman akan diarahkan sesuai dengan peta baru, selamat menggunakan!

macOS

  • sudo wget https://raw.githubusercontent.com/gvoze32/bebasid/master/releases/hosts -O /etc/hosts

Linux

  • sudo wget https://raw.githubusercontent.com/gvoze32/bebasid/master/releases/hosts -O /etc/hosts
